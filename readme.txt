== Description ==

1. This plugin adds a "Translation Theme" page to the administrator's settings menu and theme page
2. in to `page.php` e.g `t('hello page')`

== Installation ==

1.  Upload wp-translate-theme-jun/ directory to the wp-content/plugins directory.
2.  Browse to 'settings' -> 'translate' and have at it.
3.  in `page.php` e.g `t('hello page')`
